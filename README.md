# GoLang 百万级并发IM系统

## 1. 概述

实现的功能有：

**基本功能**

- 单聊、群聊发送文字、表情、图片、语音、视频等等
- 访客模式，点对点私聊，广播，机器人等
- 心跳检测下线
- 快捷回复
- 撤回消息
- 拉黑、删除

**高阶功能**

- 语音房、直播房

**技术栈**

- gin、gorm、swagger、logrus auth等中间件
